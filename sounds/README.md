# Звуковые файлы для CS2 Case Opening

Для корректной работы сайта необходимо добавить следующие звуковые файлы в папку `sounds`:

## Необходимые файлы:

1. **spin.mp3** - Звук прокрутки рулетки (тикающий звук)
2. **win.mp3** - Звук выигрыша обычного предмета
3. **rare_win.mp3** - Звук выигрыша редкого предмета (covert/rare)
4. **sell.mp3** - Звук продажи предмета (звон монет)
5. **click.mp3** - Звук клика по кнопкам
6. **case_open.mp3** - Звук открытия кейса (опционально)
7. **background.mp3** - Фоновая музыка (опционально)

## Где найти звуки:

Вы можете найти звуки CS2 на следующих ресурсах:
- https://www.sounds-resource.com/
- https://www.zapsplat.com/
- YouTube (можно конвертировать)
- Извлечь из игры CS2

## Примечание:

Если звуковые файлы отсутствуют, сайт будет работать без звуков (ошибки будут игнорироваться).
